# Criminal Law Appeal Case Management

A web application to assist with criminal appeals in Australia. Supports all criminal offence types across all Australian states and territories.

## Features
- Document Management with OCR
- AI-Powered Analysis (Timeline, Grounds of Merit)
- Case Comparison & Statistics
- Report Generation
- PayPal Integration

## Tech Stack
- Frontend: React, Tailwind CSS, Shadcn/UI
- Backend: FastAPI, MongoDB
- AI: OpenAI GPT-4o via Emergent LLM Key
